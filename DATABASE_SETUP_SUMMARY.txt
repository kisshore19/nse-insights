## 💡 Tips & Tricks

### If you forget MySQL credentials:
```sql
-- Reset MySQL root password:
-- On Windows: Use MySQL Installer to reconfigure
-- On Linux: sudo mysql_secure_installation
```

### To backup the database:
```bash
mysqldump -u root -p"root" nse_insights > nse_insights_backup.sql
```

### To restore from backup:
```bash
mysql -u root -p"root" nse_insights < nse_insights_backup.sql
```

### To see query performance:
```sql
EXPLAIN SELECT * FROM nse_daily_price WHERE trade_date = '2025-01-15';
```
